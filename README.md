# Healthcare Analytics & Machine Learning

## Project Overview

This project analyzes healthcare data using Python, Pandas, Matplotlib, and Scikit-Learn.

The objective was to perform data analysis, generate business insights, create visualizations, and build a machine learning model to predict patient cost categories.

## Dataset

* Records: 55,500
* Features: 15
* Domain: Healthcare

## Key Findings

* Diabetes generated the highest revenue in the dataset.
* Age was the most important factor influencing patient costs.
* Medical Condition and Admission Type had significant impact on billing.
* Gender showed minimal impact on patient costs.

## Machine Learning Results

* Model: Decision Tree Classifier
* Accuracy: 39.48%
